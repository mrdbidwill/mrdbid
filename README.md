<p>MRDBID will attempt to provide specimen collectors an organizational tool for a collection that remains in their control.</p>

<p>It also will encourage standard character descriptors that can be updated to provide consistent specimen comparisons.</p>

<p>Along the way, it is hoped specimens positively identified by dna procedures can be entered with accurate characters so that more positive identifications can be made in every collector's specimens through database comparisons.</p>

<p>The website uses Laravel, php, and mysql currently which are well known, widely used, and easily available tools. The open source code is available through GitHub to any who want to get a copy for their own use or to improve this common code.</p>

<p>Many thanks to the open source community and extremely helpful websites, such as stackoverflow ( https://stackoverflow.com/) and Jeffrey Way's Laracast ( https://laracasts.com/).</p>
